\begin{abstract}
Le but de ce projet est de se familiariser avec les systèmes de chiffrements à trappes. Les deux systèmes de chiffrement \textit{SCEX\_T} qui nous sont présentés sont contiennent une trappe. Cela permet de les casser plus rapidement qu'en utilisant un méthode de type force brute. Ces deux cas sont inspirés de faits réels.\\
Le travail consiste à :
\begin{itemize}
 \item Implémenter les systèmes \textit{SCEX\_T1} et \textit{SCEX\_T2}.
 \item Analyser ces systèmes et identifier la trappe (analyse mathématique).
 \item Trouver une méthode afin d'exploiter cette trappe puis l'implémenter pour une attaque.
 \item Retrouver les textes clairs à partir des chiffrés fournis.
\end{itemize}
\end{abstract}
